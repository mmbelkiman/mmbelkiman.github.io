Rede Social Gludum

Rede social voltada a desenvolvedores de jogos eletrônicos, podendo trocar experiências, criar um perfil público e divulgar os jogos criados.

Não foi utilizado nenhum sistema pré-pronto na sua confecção, todo o projeto foi construído do zero.

Sua criação partiu de pesquisas de mercado, necessidades do cliente, criação de casos de uso, documentação de banco de dados, pattern de programação.

O projeto está em constantes mudanças para atender melhor os usuarios.

Ano: 2015

Clickideia: Ferramenta exercicios educativos

Ferramenta desenvolvida juntamente com a equipe do clickideia. Através desta ferramenta é possível criar exercícios, alocar pessoas para realizar os exercícios, corrigir as respostas, gerar gráficos.

Entre os seus requisitos foi solicitado que o sistema funcionasse mesmo em caso de instabilidade de conexão web quando o aluno estivesse realizando uma lista de exercícios (cache de respostas), controle de foco de pagina, controle de tempo limite para realização das listas, período de realização e possibilidade de compartilhar exercícios com outros usuários.

Ano: 2015


JS
SQL
PHP

URL: www.clickideia.com.br
